Introduction

Buttons are used primarily on action items. Some examples include Add, Save, Delete, Sign up. Do not use Buttons as navigational elements. Instead, use Links because it takes the user to a new page and is not associated with an action. Each page may have one to two primary buttons. Any remaining calls-to-action are represented as secondary buttons.

Buttons express what action will occur when the user clicks or touches it. Buttons are used to initialize an action, either in the background or foreground of an experience.

import SpinnerService from "./spinner.js";
var spinner = SpinnerService.getSpinnerInstance();

var btnStartSpinner = document.getElementById('btnStartSpinner');
if (htmlUtils.isElementFound(btnStartSpinner)) {
    btnStartSpinner.addEventListener("click", function (event) {
        spinner.start();
        setTimeout(function () {
            spinner.stop();
        }, 4000);
    });
}
var btnStartSpinnerCustom = document.getElementById('btnStartSpinnerCustom');
if (htmlUtils.isElementFound(btnStartSpinnerCustom)) {
    btnStartSpinnerCustom.addEventListener("click", function (event) {
        spinner.start(undefined, 'Please Wait...');
        setTimeout(function () {
            spinner.stop();
        }, 4000);
    });
}
